# SmartCV - AI 简历生成器

基于 Web + AI 的智能简历生成工具，自动将无格式简历转换为精美的 LaTeX 排版简历。

## 功能特性

- 📝 支持粘贴文本或上传 PDF/Word 文件
- 🤖 AI 智能解析简历内容
- 📄 自动生成 LaTeX 代码
- 👁️ 实时 PDF 预览
- ✏️ 支持手动编辑 LaTeX 代码

## 技术栈

- Next.js 16 + React 19
- TypeScript
- Tailwind CSS
- LaTeX

## 快速开始

安装依赖：
\`\`\`bash
npm install
\`\`\`

运行开发服务器：
\`\`\`bash
npm run dev
\`\`\`

访问 http://localhost:3000
